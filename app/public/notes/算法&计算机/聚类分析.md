系统聚类法
1.写出距离矩阵
2.算出最小聚类

假设有五个样品，值分别为1，2，6.5，7，11
距离举证为
$H_0$为，代表值和值之间的距离

|     | G1  | G2  | G3  | G4  | G5  |
| --- | --- | --- | --- | --- | --- |
| G1  | 0.0 |     |     |     |     |
| G2  | 1.0 | 0.0 |     |     |     |
| G3  | 5.5 | 4.5 | 0.0 |     |     |
| G4  | 6.0 | 5.0 | 0.5 | 0.0 |     |
| G5  | 10.0| 9.0 | 4.5 | 4.0 | 0.0 |
3，4距离最近，先把它们分为一类，变为

|      | G1   | G2  | G3，4 | G5  |
| ---- | ---- | --- | ---- | --- |
| G1   | 0.0  |     |      |     |
| G2   | 1.0  | 0.0 |      |     |
| G3，4 | 6    | 5   | 0    |     |
| G5   | 10.0 | 9.0 | 4.5  | 0.0 |

可以发现1，2最近将12归为一类
则有（最短距离法）
算类与类的距离，可以有很多多算法，可以按照其中点与点最短的距离当做类与类的距离

|      | G1，2 | G3，4 | G5  |
| ---- | ---- | ---- | --- |
| G1，2 | 0    |      |     |
| G3，4 | 4.5  | 0    |     |
| G5   | 9    | 4    | 0.0 |
发现3，4和5最为接近，所以3，4，5为一类

最终，将1，2和3，4，5归为一类。

可以通过规定类的数量来确定最终截至的时间

最长聚类法和最短聚类法和瓦尔德法
1.最长距离回比最短距离类之中点与点的距离会更加散，受异常点影响比较大
2.瓦尔德法是新增一个点，使类之中的总体距离方差最小化，出来的类之间的点与点之间的距离更紧密

k-means法
直接给出分类的个数k
随机在坐标点中生成k个点，将离点最近的点归为哪一类，之后再将同类的点的中心算出，得出新的中心点
重复上述过程
直到核心点稳定。
