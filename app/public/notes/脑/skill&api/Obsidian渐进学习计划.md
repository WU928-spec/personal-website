# Obsidian 渐进学习计划

## 学习策略
采用 **A+C 方案**：
1. 先建立全库知识地图（知道有什么）
2. 再按高价值目录逐步精读（每天学一点）
3. 不追求一次学完，而是长期增量吸收

## 当前优先目录
1. `脑/`
2. `日常信息/`（优先通过脑入口间接学习）
3. `日程管理/`（优先通过脑入口间接学习）
4. `openclaw/`
5. `agentsskill/`
6. 与收支/投资/实习相关的目录

## 每日节奏
- 每天中午 12 点后第一次 heartbeat：
  - 学习一个小模块
  - 更新学习记录
  - 判断是否值得形成新 skill 或补充旧 skill
- 不做暴力全库扫读
- 新结论优先写入 `脑/skill&api/`

## 今日原则
- 每次只推进一点
- 优先高频、稳定、可复用内容
- 临时笔记只建索引，不强行写入长期记忆
