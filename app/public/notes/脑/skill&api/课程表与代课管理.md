# 课程表与代课管理

## 适用场景
当用户询问以下内容时应优先加载本 skill：
- 某天某节有没有课
- 哪些是自己的课，哪些是代课
- 根据课程表安排时间
- 读取课程表 Excel 并解释颜色、合并单元格、地点信息

## 核心知识
- 课程表 Excel 可能包含**合并单元格**，不能只用 pandas 粗读
- 应优先用 **openpyxl** 检查：
  - 合并区域
  - 单元格颜色
  - 原始单元格位置
- 颜色语义：
  - 绿色：自己的课
  - 蓝色：长期代课
  - 黄色：短期代课
- 六教 / 二教 / 五教 等表示教学楼，后续数字表示房间号

## 推荐流程
1. 先定位 Excel 文件位置
2. 读取 sheet 名称与原始布局
3. 检查合并单元格
4. 检查颜色
5. 将时间、节次、星期、课程、地点统一解释
6. 回答用户“某时段是否有课”时，明确：
   - 节次
   - 时间
   - 课程
   - 类型（自己课 / 长期代课 / 短期代课）
