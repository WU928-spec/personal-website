# 细则-应收、其他应收、预收、应付、其他应付（4.28）

# \*8一、类别区分

应收账款和其他应收款都属于资产，是别人欠我

预收款项、应付账款、其他应付款，属于负债，是我欠别人

# 二、定义

## 1、应收账款（经营性应收）

### (1)属性：流动资产

### (2)核心定义：

企业日常主营业务销售商品、提供劳务，应向客户收取的款项。

### (3)适用场景：

卖货赊销、工程服务未回款、主营业务产生的欠款。

### (4)财务附注重点披露：

账龄结构（1 年以内 / 1-2 年 / 3 年以上）

坏账准备计提政策、计提比例、本期计提 / 转回金额

前五大欠款方名称、欠款金额、占比

坏账准备

## 3、预付账款

按款项性质：材料款、设备款、工程款拆分

## 6、预收账款

### （1）属性：流动负债

### （2）核心定义：

提前收到客户货款 / 服务费，但商品未发货、服务未提供，属于欠客户的 “履约义务”。

### （3）适用场景：

预收货款、预收款、会员预存费、项目预付款。

新准则说明：

执行新收入准则企业，主营业务预收款统一计入「合同负债」；

预收账款现在多用于租赁、老旧核算口径的预收款项。

### （4）财务附注重点披露：

期末预收款项对应项目、履约进度

期后结转收入金额
