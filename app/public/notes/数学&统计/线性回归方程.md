**一元线性回归** 的模型形式为：  
$y = \beta_0 + \beta_1 x + \varepsilon$
### 1. 回归系数 \(\beta_0\) 和 \(\beta_1\) 的计算

**样本公式**（最小二乘法估计）：
$\hat{\beta}_1 = \frac{\sum_{i=1}^{n} (x_i - \bar{x})(y_i - \bar{y})}{\sum_{i=1}^{n} (x_i - \bar{x})^2}$
$\hat{\beta}_0 = \bar{y} - \hat{\beta}_1 \bar{x}$
其中：
- $\bar{x}$是x的样本均值
- $\bar{y}$是$y$的样本均值
- n是样本量

---

### 2. 决定系数 \(R^2\) 的计算

$R^2 = \frac{SSR}{SST} = 1 - \frac{SSE}{SST}$
其中：
- **总平方和**：  
$SST = \sum_{i=1}^{n} (y_i - \bar{y})^2$
- **回归平方和**：  
$SSR = \sum_{i=1}^{n} (\hat{y}_i - \bar{y})^2$
- **残差平方和**：  
$SSE = \sum_{i=1}^{n} (y_i - \hat{y}_i)^2$
并且满足
SST = SSR + SSE
**注意**：在一元线性回归中，\(R^2\) 也等于样本相关系数 \(r\) 的平方：
$R^2 = r^2 = \left[ \frac{\sum (x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum (x_i - \bar{x})^2 \sum (y_i - \bar{y})^2}} \right]^2$


---


2.  **计算公式**：  
    SEE 的计算公式为：
    $\text{SEE} = \sqrt{\frac{\sum_{i=1}^{n} (Y_i - \hat{Y}_i)^2}{n - k - 1}}$=$\sqrt{\frac{SSE}{n - k - 1}}$
    - $Y_i$：第 $i$个观测点的实际值
    - $\hat{Y}_i$：第 i 个观测点的预测值
    - n：样本量
    - k：模型中自变量的个数
    - n - k - 1是 **残差自由度**，用于对模型参数数量进行调整，避免高估精度。
