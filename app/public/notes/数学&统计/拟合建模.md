# 连续性函数拟合
## 数据预处理
### 缺失值处理
1.完全随机缺失([MACR](MACR.md))：缺失较少(低于5％)，直接删除整个个例
2.随机缺失([MAR](MAR.md))：[多重插补](多重插补.md)或者[极大似然估计](极大似然估计.md)或者[特殊值代替](特殊值代替)
3.非随机缺失([MNAR](MNAR.md))：需要结合经验分析，MNAR产生的原因，再进行处理。

## 模型评估
1. **回归拟合**：
    - 核心指标：[R^2](R^2.md)（解释力）、[RMSE](RMSE.md)（直观误差）；
    - 数据含异常值：用 [MAE](MAE.md) 替代 RMSE；
    - 多变量回归：用调整[AdjustedR^2](AdjustedR^2.md) 替代 R^2。
2. **分类拟合**：
    - 类别平衡：[准确率](准确率.md) + [F1](F1.md)；
    - 类别不平衡：F1（宏平均） + [AUC](AUC.md)；
    - 需解释错误类型：绘制[混淆矩阵热力图](混淆矩阵热力图.md)。
3. **时间序列拟合**：
    - 误差评估：RMSE + [sMAPE](sMAPE.md)；
    - 时序依赖评估：残差 [ACF/PACF](ACF&PACE.md) + 时序对比图；
    - 形状相似度：DTW（时序长度不匹配时）。
