$若二元函数  f(x, y) 当 (x, y) \to (a, b) 时的极限为  L ，且 L > 0 （或  L < 0)$ ，$则存在 (a, b) 的一个去心邻域，使得在该邻域内 f(x, y) > 0 （或 f(x, y) < 0 ）。$

$仅以  L > 0 的情况为例，L < 0 的情况类似可证。$$取 \varepsilon = \frac{L}{2} ，根据极限的定义，存在  \delta > 0 ，使得当 0 < \sqrt{(x - a)^2 + (y - b)^2} < \delta 时，有：$
$|f(x, y) - L| < \frac{L}{2}$,$f(x, y) > L - \frac{L}{2} = \frac{L}{2} > 0$
这表明在 \( (a, b) \) 的去心邻域 $0 < \sqrt{(x - a)^2 + (y - b)^2} < \delta 内，f(x, y) > 0 。$
