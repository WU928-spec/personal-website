中心极限定理是概率论中最重要的定理之一。它指出，在适当的条件下，大量独立随机变量的和的分布会趋近于正态分布，无论这些随机变量原本的分布是什么。

**定理（独立同分布情形的林德伯格-莱维中心极限定理）**

设$X_1, X_2, X_3, \dots, X_n$是一系列独立同分布的随机变量，具有有限的期望值$\mu$和有限的方差$\sigma^2 > 0$。

定义这些变量的样本均值为：
$\bar{X}_n = \frac{1}{n} \sum_{i=1}^{n} X_i$
定义这些变量的标准化和（或标准化样本均值）为：
$Z_n = \frac{\bar{X}_n - \mu}{\sigma / \sqrt{n}} = \frac{\sum_{i=1}^{n} X_i - n\mu}{\sqrt{n} \sigma}$
那么，当$n \to \infty$时，随机变量$Z_n$的分布收敛于标准正态分布。即对于任意实数z，
$\lim_{n \to \infty} P(Z_n \le z) = \Phi(z)$
其中$\Phi(z)$是标准正态分布的累积分布函数。

---

### 证明

**证明步骤：**

1.  **设定与目标**
    设 $Y_i = \frac{X_i - \mu}{\sigma}$。则$Y_i$ 是标准化后的随机变量，满足$E[Y_i] = 0$ 和$\text{Var}(Y_i) = 1$。
    我们的目标变量$Z_n$可以写为：
    $Z_n = \frac{1}{\sqrt{n}} \sum_{i=1}^{n} Y_i$
    我们需要证明$Z_n$的特征函数收敛于标准正态分布的特征函数$e^{-t^2/2}$。

2.  **[特征函数](特征函数.md)的定义与性质**
    随机变量U的特征函数定义为$\phi_U(t) = E[e^{itU}]$，其中i是虚数单位。
    - 独立随机变量和的特征函数等于各变量特征函数的乘积：$\phi_{U+V}(t) = \phi_U(t) \phi_V(t)$。
    - 如果 $\phi_{U_n}(t)$逐点收敛于$\phi_U(t)$，且$\phi_U(t)$在t=0处连续，则$U_n$的分布收敛于U的分布（Lévy连续性定理）。

3.  **计算$Z_n$的特征函数**
    由于$Y_i$是独立同分布的，$Z_n$的特征函数为：
    $\phi_{Z_n}(t) = E[e^{itZ_n}] = E\left[ \exp\left( it \cdot \frac{1}{\sqrt{n}} \sum_{k=1}^{n} Y_k \right) \right] = \left( E\left[ \exp\left( i \frac{t}{\sqrt{n}} Y_1 \right) \right] \right)^n = \left[ \phi_Y\left( \frac{t}{\sqrt{n}} \right) \right]^n$
    其中$\phi_Y(t)$是$Y_i$的特征函数。

4.  **对$\phi_Y(t)$进行泰勒展开**
    在t=0处对$\phi_Y(t)$进行泰勒展开。回忆$E[Y] = 0$，$E[Y^2] = \text{Var}(Y) = 1$。
    $\phi_Y(t) = E[e^{itY}] = E\left[ 1 + itY + \frac{(itY)^2}{2!} + o(t^2) \right] = 1 + itE[Y] - \frac{t^2}{2}E[Y^2] + o(t^2)$
    代入$E[Y] = 0$和$E[Y^2] = 1$：
    $\phi_Y(t) = 1 - \frac{t^2}{2} + o(t^2)$
    这里的$o(t^2)$表示当$t \to 0$时，余项比$t^2$更快地趋近于零。

5.  **代入并求极限**
    现在我们将$t/\sqrt{n}$代入上式：
    $\phi_Y\left( \frac{t}{\sqrt{n}} \right) = 1 - \frac{t^2}{2n} + o\left( \frac{t^2}{n} \right)$
    因此，$Z_n$的特征函数为：
    $\phi_{Z_n}(t) = \left[ 1 - \frac{t^2}{2n} + o\left( \frac{t^2}{n} \right) \right]^n$

6.  **应用极限公式**
    我们知道一个重要的极限：$\lim_{n \to \infty} \left( 1 + \frac{a}{n} + o(1/n) \right)^n = e^a$。
    在我们的表达式中，令$a = -t^2/2$，则有：
    $\lim_{n \to \infty} \phi_{Z_n}(t) = \lim_{n \to \infty} \left[ 1 - \frac{t^2}{2n} + o\left( \frac{1}{n} \right) \right]^n = e^{-t^2/2}$

7.  **得出结论**
    我们证明了$Z_n$的特征函数$\phi_{Z_n}(t)$收敛于标准正态分布的特征函数$e^{-t^2/2}$。
    根据 Lévy 连续性定理，可以得出结论：$Z_n$的分布收敛于标准正态分布。
    $\boxed{\lim_{n \to \infty} P(Z_n \le z) = \Phi(z)}$
