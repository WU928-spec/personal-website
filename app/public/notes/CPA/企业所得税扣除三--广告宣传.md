企业筹建期间，与筹办活动有关的业务招待费支出，按实际发生额的60%计入企业筹办费。（筹建期中无收入，所以按照60计入筹办费用）

| 行业类型                     | 扣除限额                  |
| ------------------------ | --------------------- |
| 一般企业                     | 不超过当年销售（营业）收入 **15%** |
| 化妆品制造/销售、医药制造、饮料制造（不含酒类） | 不超过当年销售（营业）收入 **30%** |
| 烟草企业                     | **不得扣除**              |
- 超过部分准予在以后纳税年度结转扣除

广告宣传看业务实质，是否起到宣传推广作用
筹建期，广告费用全额计入筹办费

| 费用类型                    | 扣除限额             | 超限部分处理    |
| ----------------------- | ---------------- | --------- |
| **公益性捐赠**               | 利润总额的 **12%**    | **3年内**结转 |
| 广告费和业务宣传费（一般企业）         | 销售收入的 **15%**    | **无限期**结转 |
| 广告费和业务宣传费（化妆品/医药/饮料）    | 销售收入的 **30%**    | **无限期**结转 |
| 业务招待费                   | 发生额的60%，且≤销售收入5‰ | **不得结转**  |
| 职工福利费                   | 工资总额的 **14%**    | **不得结转**  |
| 工会经费                    | 工资总额的 **2%**     | **不得结转**  |
| 职工教育经&#x8D39;**（一般企业）** | 工资总额的 **8%**     | **无限期**结转 |
