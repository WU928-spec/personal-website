价内税价外税概念区分：
价内税是计包含在价格里的税，消费税是价内税，所以产品真正的价格是成本+利润+消费税=真正价格
成本+利润+真正价格×消费税税率=真正价格，所以真正价格=（成本＋利润）/1+消费税税率
价外税是不包含在产品本本身的价格中的，增值税是价外税，所以增值税=不含税价格×增值税税率，所以含税价格=不含税价格（1+增值税税率），所以不含税价格=含税价格/（1+增值税税率）这里的不含税价格就是上面的真正价格，所以计算消费税是首先要计算出真正价格也就是不含税价格。

连续生产不可抵扣消费品情况：
1.酒类（除葡萄酒）
2.高档消费品
3.特殊收拾
4.溶剂油：外购溶剂油生产成品油，不能抵扣

委托加工应税消费品的计税依据首先以受托类同类消费品的销售价格确定。

委托加工的成品售出的同类价格作为计税依据，若没有，用材料成本+加工费计算。

委托加工行为中，委托方将收回的应税消费品，以不高于受托方的计税价格出售的，不缴纳消费税

消费税在移交加工公司时，已经代缴，若是价格低，不需要缴费，高的话要补缴



# 消费税总分机构纳税地点（一般情形）笔记

## 一、核心原则

📌 分别申报为原则，批准汇总为例外（区别于卷烟批发企业的“总机构直接汇总、无需审批”）

## 二、核心政策依据

1. 《中华人民共和国消费税暂行条例实施细则》第二十四条：总分机构不在同一县（市），分别向各自机构所在地申报；经财政部、税务总局或其授权机关批准，可由总机构汇总申报。

2. 同一省级范围内汇总：需经省级财政厅（局）、税务局审批（依据财税〔2012〕42号）。

## 三、不同情形具体处理（重点）

| 总分机构情形           | 纳税地点                | 关键备注     |
| ---------------- | ------------------- | -------- |
| 不在同一县（市），跨省级行政区  | 各自向机构所在地申报          | 无批准，不得汇总 |
| 不在同一县（市），同一省级行政区 | 批准后总机构汇总申报；未批准则分别申报 | 审批权限在省级  |
| 在同一县（市）          | 总机构统一向当地申报          | 无需额外批准   |

## 四、与卷烟批发企业特殊规则对比（易混点）

| 对比项目  | 一般应税消费品（本次笔记重点） | 卷烟批发企业（特殊规则）   |
| ----- | --------------- | -------------- |
| 核心规则  | 分别申报为原则，批准汇总为例外 | 总机构直接汇总申报，无需审批 |
| 跨地区处理 | 跨省级必须分别申报       | 跨地区均由总机构申报     |
| 审批要求  | 需经财政、税务部门批准     | 无任何审批要求        |

## 五、补充：一般消费税其他纳税地点（拓展）

- 销售/自产自用应税消费品（无特殊规定）：机构所在地/居住地申报
    
- 委托加工（受托方非个人）：受托方机构所在地/居住地解缴税款；受托方为个人，委托方自行申报
    
- 进口应税消费品：报关地海关申报
    

## 六、总结（必记）

一般情形下，消费税总分机构纳税需严格遵循“分别申报+批准汇总”，核心是“无批准不汇总”；与卷烟批发企业的核心差异的是：是否需要税务机关审批。


企业所得税纳税人：
居民企业：指依法在中国境内成立，或者依照外国（地区）法律成立但实际管理机构在中国境内的企业。其中，“实际管理机构”是指对企业的生产经营、人员、账务、财产等实施实质性全面管理和控制的机构。
非居民企业：指依照外国（地区）法律成立且实际管理机构不在中国境内，但在中国境内设立机构、场所的，或者在中国境内未设立机构、场所，但有来源于中国境内所得的企业。其中，“机构、场所”包括管理机构、营业机构、办事机构、工厂、开采自然资源的场所、提供劳务的场所等，非居民企业委托营业代理人从事生产经营活动的，该营业代理人视为其设立的机构、场所。

纳税义务关联：居民企业需就其来源于中国境内、境外的所得缴纳企业所得税；非居民企业则根据是否设立机构、场所，以及所得与机构、场所是否有实际联系，确定不同的纳税义务范围。

特殊情况：个人独资企业和合伙企业不缴纳企业所得税
