| 类型          | 核心特点                            | 关键词                 |
| :---------- | :------------------------------ | :------------------ |
| **防御型战略组织** | 专注现有市场和产品，追求稳定和效率，很少创新          | 守住现有业务，不冒险扩张        |
| **开拓型战略组织** | 不断寻找新机会、开发新产品 / 新市场，以创新为核心      | 主动探索前沿、追求新机会、扁平创新   |
| **分析型战略组织** | 平衡 “稳定” 与 “创新”，一边守住原有业务，一边跟进新机会 | 双线作战：传统业务 + 新业务同时推进 |
| **反应型战略组织** | 被动应对环境变化，缺乏稳定战略，调整混乱            | 被动、无固定模式、反应滞后       |

核心知识点 
1. 战略先行：企业先调整发展战略，再适配组织结构
2. 结构滞后性：组织结构具有稳定性、惯性，调整改造需要时间、成本，存在**结构滞后效应**
3. 战略变化快，组织结构变化慢
