工艺升级 
**代表企业：美的集团** 
- 通过**工业互联网、IoT设备数据采集、AI排产、预测性维护**优化生产线。
- 利用数据科学优化生产节拍、降低能耗、减少次品率，实现更高效、更低成本的制造流程。
产品升级 
**代表企业：海尔智家**
- 基于**用户使用大数据、家庭场景数据分析**，从传统家电升级到智能家电、全屋智能方案。
- 用数据洞察用户习惯，迭代产品功能，推出更高附加值的智能化产品。
功能升级
**代表企业：申洲国际** 
- 从单纯服装代工（低附加值），利用**销售数据、用户画像、供应链数据**，转向自主设计、品牌运营与渠道掌控。 
- 实现从OEM代工向研发、品牌等高附加值环节跃迁。
链条升级（跨产业升级） 
**代表企业：三一重工** 
- 从卖工程机械，转向利用**设备运行大数据、AI运维、根云工业互联网平台**，对外提供数字化服务。
- 跳出传统制造价值链，进入工业互联网与数字服务新赛道。 

| 战略类型 | 适用场景                | 核心行为                        |
| ---- | ------------------- | --------------------------- |
| 防御者  | 全球化压力低，优势资源仅适合本土    | 聚焦本土客户、强化分销、定制本土产品          |
| 扩张者  | 全球化压力低，优势资源可复制到相似市场 | 向与本国市场相似的海外市场扩张             |
| 躲闪者  | 全球化压力高，优势资源难转移      | 与跨国企业合作、转向细分市场、剥离业务         |
| 抗衡者  | 全球化压力高，优势资源可转移      | 全球竞争、正面进攻，通过技术 / 成本优势对抗跨国企业 |
|      |                     |                             |
