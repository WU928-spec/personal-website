# 期权价格评估（如何估计期权的价值）
  
- 复制原理
    
    - 通过构建一套股票和负债的组合使得其的涨跌幅度和期权价格相同，再通过估计股票和负债的成本估计期权价格  
        
    
    - 方法：由涨跌幅相同得股票上行市价-借款本利和=期权上涨价格；股票下行市价-借款本利和=期权下跌价格股票上行市价−借款本利和=期权上涨价格；股票下行市价−借款本利和=期权下跌价格股票上行市价-借款本利和=期权上涨价格；股票下行市价-借款本利和=期权下跌价格股票上行市价−借款本利和=期权上涨价格；股票下行市价−借款本利和=期权下跌价格。以看涨期权举例推导  
        
    
    - 股票上行市价=股数H×单股上涨价格，同理股价下行市价=股数H×单股下跌价格。联立方程，可以观察到，这是一条由两点确定的直线方程，两点分别是（单股上涨价格，期权上升价格），（单股下跌价格，期权下降价格）。所以斜率H=期权上升-期权下降/股价上升-股价下降。之后再解出借款价格就可以了。最后期权价格是：购买股票价格-借款价格。
- 风险中性原理  
    
    - 风险中性原理假设所有资产的期望报酬率都等于无风险报酬率（这里指的是数学期望）  
        
    
    - 所以这里假设期权价格的收益率=无风险报酬率r，期望报酬率=（上行概率×上行期权价值+下行概率×下行期权价值）/1+r，因为默认所有的期望一样，所以可以用股票期望价值估计期权价值，在这里的股票价格要和概率的起始点价格一样。  
- 二叉树期权定价模型  
    
    - 多步定价，下图：![](https://api2.mubu.com/v3/document_image/29332854_5c5272f8-a43c-4337-9137-ea7e76f26bbe.png)  
        
    
    - 通过风险中性概率p构建股票的期望二叉树。  
        
    
    - 通过二叉树最低端的盈利和亏损价格作为期权期望价值，再通过最低端的期望期权价值逐步往二叉树上回溯得到期权价值，需要乘无限复利系数折现回归。![](https://api2.mubu.com/v3/document_image/29332854_2de64f93-f811-493a-b843-7897f8e382d8.png)

# 企业价值评估（现金流量折现模型）
- 现金流量模型  
    
    - 未来现金流量现值=$\sum_{n=1}^{\infty} \frac{实体现金流量}{1 + （加权平均资本成本）^n}$（企业价值等于未来现金流量现值和清算价值中最高的一个）
        
    
    - 实体现金流量=EBIT（1-T）-净现金经营性资产增加（EIBT是息税前利润，T是税率）  
        
    
    - 实体现金流量=债务现金流量+股权现金流量，债务现金流量=税后利息收入-净负债增加 股权流=净利润-股东权益增加（股东可以看作是债务人），计算什么价值就用什么流量  
        
    
    - 永续增长情况下的实体价值=下期实体现金流量/（加权平均资本成本-永续增长率）（从现在开始永续增长，下期才作用到现金流量上）  
        
    
    - 两段式增长模型：前面正常算，后面用永续算，记得用复利系数折现。
