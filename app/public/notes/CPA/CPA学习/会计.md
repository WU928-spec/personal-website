- # 会计要素  
    - 资产负债表（描述企业财务情况）  
        - 资产  
            - 性质：过去事项形成，预期带来资源流入企业，可货币计量  
        - 负债  
            - 性质：过去事项所形成的义务，预期带来资源流出企业，可货币计量。  
        - 所有者权益  
            - 所有者权益，也称为股东权益或净资产，是企业资产中扣除负债后的剩余权益，代表了所有者（股东）对企业资产的最终所有权。  
            - 构成  
                - 投入资本（股东投入的资本和外界投入的资本）  
                    - 实收资本  
                        - 企业从股东处实际收到的资金总额，包括现金和非现金资产。  
                    - 资本公积  
                        - 资本公积是指企业因资本溢价、资产评估增值、接受捐赠、政府补助等原因形成的权益，是企业所有者权益的重要组成部分。  
                            [资本溢价情况详解](股票面值和发行价值.md)
                - 留存收益（净利润中未分配给股东的一部分）  
                    - 未分配的利润
                        - 属于企业在日常经营活动中产生，但与主营业务无关的一部分  
                    - 盈余公积  
                        - 主营业务部分的净利润 
    - 利润表（描述企业经营情况）  
        - 收入  
            - 日常活动，和投入资本无关的收入  
        - 费用  
            - 日常活动，和投入资本无关的支出  
        - 利润  
            - 收入-费用+利得-损失  
    - 利得
        - 定义：偶然性的收入净额  
        - 种类  
            - 其他综合收益（记入所有者权益）  
                - 用于记录暂时性和非经常性的利得或损失，记入所有者权益，避免对当期利润产生影响  
            - 营业外收入（记入当期利润表）  
                - 用于记录i偶发性和一次性的利得和损失，一般是营业中的意外收入，记录进当期利润表，以代表正常经营情况  
    - 损失  
        - 定义：偶然性的支出净额  
            - 详见利得描述的
- # 存货初始计量  
    - 存货定义：企业正常经营活动的损耗品  
    - 存货认定：  
        - 1.持有意图是为了损耗后盈利  
        - 2.拥有所有权  
        - 3.[特殊情况](存货认定特殊情况.md)（以后具体写）  
    - 存货的采购成本  
        - 存货的采购成本包括购买价款、相关税费、运输费、装卸费、保险费、合理损耗、挑选整理费以及其他直接费用。这些费用共同构成了存货的实际成本，反映了企业为获取存货所付出的总代价。（相关税费不包括企业增值税）  
            - [一般纳税人](一般纳税人和小规模纳税人.md)不用交企业[增值税](增值税.md)  
            - [小规模纳税人](一般纳税人和小规模纳税人.md)要交增值税  
		    - 注意点：存货入库前的损耗记入存货的采购价值，入库后损耗记入当期损益  
    - 数据资源可被记入存货，存货的成本为进行数据清理等工作的成本。
	- ### 商品流通企业采购商品的进货费用处理思路
		- 如果进货费用金额较大，或者与商品直接相关，这些费用会直接计入商品成本。这样可以更准确地反映商品的实际成本。
		- 如果进货费用金额较大或需要进一步分配，可以先将这些费用归集到一个专门的账户中，比如“进货费用”账户。
			- 如果商品尚未出售，归集的进货费用会作为存货成本的一部分，计入期末存货成本。这样可以确保存货的价值反映其实际成本。
			- 如果商品已经出售，归集的进货费用会从存货成本中转出，计入主营业务成本。这样可以准确反映当期的销售成本和利润。
		- 如果进货费用金额较小，或者与商品的直接关系不明显，这些费用可以直接计入当期损益，作为销售费用处理。这样可以简化会计处理，减少核算的复杂性。
- # 存货的计量方法
	- 计量方法  
	    - 实际成本  
	        - 个别计价法  
	            - 使用的每个存货以其进货价格单独计价（缺点：工作量大 优点：精确）  
	        - 先进先出法  
	            - 使用的每个存货以其进货时的顺序出货计价。（存货单价变动大的情况下会很复杂）  
	            - 当物价持续上升时，由于出货时账面上是按照低的出货，但实际上高低都有，所以会高估库存存货价值和企业当期利润  
	            - 同理当物价持续下降时，会低估库存存货价值和企业当期利润  
	        - 月末一次加权平均法  
	            - 在月末对库存存货取加权平均价格，出货时按照平均价格计价（优点：方便 缺点：不清楚每个存货的单独计价，不利于企业管理）  
	        - 移动加权平均法  
	            - 每次出货时都计算现有库存的加权平均，再计算出货价格。（优点：每月计算，更加客观，更利于企业管理。缺点：当库存单价变化量大时，计算量大）  
		    - [不同方法的使用情况](存货的实际计量方法的使用辨析.md)
	    - 计划成本  
	        - 按照计划采购的成本计价（不客观）  
	- 存货成本的结转  
	    - 对外销售商品  
	        - 借主营业务成本  
	        - 贷库存商品 
	        - 贷[存货跌价准备](存货跌价准备.md)  
	        - 借主营业务成本  
	    - 对外销售原材料  
	        - 借其他业务成本  
	        - 贷原材料  
	        - 贷存货跌价准备  
	        - 借其他业务成本  
	    - 周转材料的会计处理（包装物） 
        - 生产时领用的包装物  
            - 借生产费用  
            - 贷周转材料——包装袋  
        - 销售时不单独计价的包装物  
            - 借销售费用  
            - 贷周转材料——包装袋  
        - 若包装袋可以长期时用，贷周转材料时可以按批次计提，详情点[我](分次摊销法.md)

- # 存货期末计量  
    - 资产负债表上计  
        - 实际存货成本  
        - 可变现净值  
            - 指在将存货加工为销售品后销售市场价格减去存货到销售时需要的成本，代表了这批存货在实际销售时的价值  
        - 资产负债表计MIN(实际存货成本，可变现净值)  
        - 可变现净值计算方法  
            - 直接出售  
                - 有合同  
                    - 按合同价计算-销售费用-相关税费  
                - 无合同  
                    - 预计市场价-销售费用-相关税费  
            - 继续加工  
                - 产成品可变现净值大于成本  
                    - 直接按存货成本计算  
                - 产成品可变现净值小于成本  
                    - 产成品售价-销售费用-相关税费-进一步加工费  
        - 存货有合同部分和非合同部分  
            - 分别计算合同部分和非合同部分的成本，最后加在一起。  
    - [存货跌价准备的计提，转回，结转](存货跌价准备.md)
- # 存货清查盘点  
    - 待处理财产损益  
        - 存货盘盈盘亏在批准前后所存在的中间账目，在期末时此账目应为0  
        - 中间账目存在意义  
            - 盘盈盘亏需要审查确定，所以存在一个中间账户，避免频繁转动  
    - 存货盘盈处理  
        - 审批前  
            - 借：原材料  
            - 贷：待处理财产损益  
        - 审批后  
            - 借：待处理财产损益  
            - 贷：管理费用  
    - 存货盘亏处理  
        - 审批前  
            - 借：待处理财产损益  
            - 贷：原材料 应交增值税  
        - 审批后  
            - 借：原材料[残料]；其他应付款[赔款]；管理费用[一般经营损失]；营业外支出[自然灾害]  
            - 贷：待处理财产损益
	    - [进项税税额转出问题](增值税.md)
- # 固定资产初始计量  
    - 外购固定资产  
        - 包括购买物品到使用前的一切成本（不包括差旅费，员工培训费和可抵扣的增值税） 
    - 自建固定资产  
        - 自营方式建造固定资产  
            - 工程物资  
                - 领用时按实际成本转入所建工程成本，结束后，剩余的物资转入企业存货  
            - 原材料和库存商品  
                - 借 在建工程  
                - 贷 原材料 库存商品
            - 盘盈和盘亏  
                - 工程完成前  
                    - 盘盈和盘亏记入在建工程成本  
                - 工程完成后  
                    - 盘盈和盘亏记入当期损益  
            - 土地使用权  
                - 取得使用权确认为无形资产  
                - 建造过程占有土地使用权的摊销记入建造成本  
            - 人工辅助生产成本其他费用  
                - 记入所建工程成本  
            - [暂估入账问题](暂估入账问题.md)  
    - 出包方式建造固定资产  
        - 待摊费用是不能直接划归进某一项支出的成本，由整个项目一起承担，我们需要按比例分给各个工程  
        - 某项工程应分配的待摊支出=（该项工程的建筑工程支出+该项工程的安装工程支出+该项工程的在安装设备支出）×待摊支出分配率  
        - 固定资产的成本=建筑工程支出+应分摊的待摊支出  
        - 需要安装设备的成本=设备成本+为设备安装发生的基础，支座等建筑工程支出+安装工程支出+应分摊的待摊支出。  
    - 购买具有融资性质的固定资产  
        - 实际价格按照，后续付款按照规定利率的折现确定  
        - 在利率期间按照[实际利率法](融资资产摊销法.md)进行摊销

        - 竣工资产不需调整计提，详见[暂估入账问题](暂估入账问题.md)

- # 无形资产初始计量
	- 无性资产定义
		- 与该资产有关的经济利益很可能流入企业
		- 该无形资产的成本能可靠的计量
			- 商誉不能计量成本，所以不属于无形资产
	- 外购无形资产初始计量
		- 按照实际成本计量，即以取得无形资产到能达到预定用途的所有支出，作为无形资产成本
		- 购买具有融资属性的无形资产，需要按支付的现价确定成本，和固定资产差不多
		- 土地使用权的处理
			- 通常来说，企业安装取得所支付的价款和税费作为无形资产成本，但政府无偿划拨给企业的土地使用权按照公允价值计算成本，若不能可靠的取得，则按名义金额规定。
			- 土地使用用于自行开发建造厂房等自用地上建筑时，土地使用权作为无形资产进行核算，土地使用权与地上建筑物分别进行摊销和提取折旧。
			- 固定资产建造过程中所占用的土地使用权的摊销额应记入资产建造成本中。
			- 房地产开发企业，建造对外出售的建筑的土地使用权记入所建造房屋的建筑物成本，未开发的土地使用权作为存货。
			- 改变土地使用权的目的时，转为投资性房地产
			- 企业外购的房屋，应按照比例在房屋和土地使用权之间分配，如果不能可靠分配，那么就作为固定资产计量。

# 内部研发无形资产
- 研究阶段的处理
	- 企业内部研究阶段的支出，全部费用化，计入当期损益(管理费用)
- 开发阶段的处理
	- 开发阶段支出资本化条件(同时满足)
		- 完成该无形资产以使其能够使用或出售在技术上具有可行性
		- 具有完成该无形资产并使用或出售的意图
		- 能够确定无形资产产生利益的方式，无论是内部使用或是辅助生产
		- 有足够的能力完成无形资产的开发
		- 无形资产的支出能够可靠计量
	- 在开发阶段满足资本化条件的支出计入无形资产开发成本，不满足的费用化，计入当期损益
- 一般会计分录
	先填充研发支出的费用化和资本化支出，在无形资产完成时，分别把这两者转到管理费用和无形资产成本
# 无形资产后续计量和处置
- 无形资产摊销的会计处理
	- 使用寿命不确定
		- 不摊销
	- 使用寿命确定
		- 1.进行摊销
		- 2.残值视为0(若有交易价，则为交易价)
		- 3.可供使用当月时摊销，处置时不摊销
	- 摊销方法
		- 可以可靠了解应其收获的经济利益
			- 生产总量法
		- 不可以
			- 直线法
	- 摊销计入科目
		- 如果是管理类的无形资产
			- 摊销计入管理费用
		- 出租用的
			- 计入其他业务成本
		- 生产商品获利
			- 摊销计入相关资产成本
- 无形资产减值的会计处理
	- 借：资产减值损失
		- 贷：无形资产减值准备
		- 无形资产减值损失确认后，不得转回
- 无形资产出售，报废的会计处理
	- 无形资产的出售
		- 将取得的价款与无形资产账面的差额计入当期损益
		- 借：银行存款
		- 无形资产减值准备
		- 累计摊销
		- 无形资产
	- 无形资产的支出
	- 无形资产的报废










长期股权投资
 同一控制下 
	 存货类投资，按照被投资公允价值的百分比作为投资成本
	 股票类投资，被投资公允价值的百分比-股票发行成本
	 手续费等费用进入管理费用
 非同一控制下
	 存货类投资，按照投资公允价值作为投资成本

前面题目主要涉及长期股权投资相关知识点：
初始投资成本确定 非同一控制下企业合并- 以非货币性资产（如无形资产）为对价，初始投资成本为付出对价的公允价值加上相关税费（如无形资产增值税） 。如甲公司以公允价值2000万元、适用增值税率6%的无形资产取得丙公司70%股权，初始投资成本 = \(2000 + 2000×6\% = 2120\)万元 。 - 以发行权益性证券为对价，初始投资成本按发行权益性证券的公允价值确定，发行佣金等冲减资本公积 - 股本溢价，不构成初始成本。如甲公司发行1000万股（每股面值1元，市价5元）取得乙公司30%股权，初始投资成本为\(1000×5 = 5000\)万元 。 2. **同一控制下企业合并**：初始投资成本为合并日被合并方在最终控制方合并财务报表中净资产账面价值的份额 。如甲、乙同属一集团，乙在最终控制方合并报表净资产账面价值2500万元，甲取得乙80%股权，初始投资成本 = \(2500×80\% = 2000\)万元 。 ### 交易费用处理 1. **非同一控制下企业合并**：审计、法律服务等中介费用计入管理费用 ，并非长期股权投资成本 。 2. **同一控制下企业合并**：审计、法律咨询费等计入管理费用 ；发行债务性工具的交易费用计入债务性工具初始确认金额 。 3. **发行权益性证券**：支付的股票发行费用等先冲减溢价发行收入（资本公积 - 股本溢价 ），不足冲减依次冲减盈余公积和未分配利润 。 ### 重大影响判断 1. 通常持股比例在20% - 50% ，或虽持股比例不足但能通过派出董事、提供关键技术资料等方式对被投资单位财务和经营政策施加重大影响 ，则对被投资单位具有重大影响 。 如持有25%股权并派董事、持有18%股权但提供关键技术资料等情况。
