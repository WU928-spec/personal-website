可扣除的商业保险

| 保险类型                | 扣除条件/标准                                                |
| ------------------- | ------------------------------------------------------ |
| **1. 特殊工种人身安全保险**   | 依照国家有关规定为特殊工种职工支付的人身安全保险费，可全额扣除                        |
| **2. 补充养老保险**       | 不超过职工工资总额 **5%** 的部分准予扣除                               |
| **3. 补充医疗保险**       | 不超过职工工资总额 **5%** 的部分准予扣除                               |
| **4. 财产保险**         | 企业参加财产保险，按规定缴纳的保险费准予扣除                                 |
| **5. 雇主责任险、公众责任险**  | 企业参加雇主责任险、公众责任险等责任保险，按规定缴纳的保险费准予扣除                     |
| **6. 职工因公出差人身意外险**  | 职工因公出差乘坐交通工具发生的人身意外保险费支出准予扣除                           |
| **7. 税优健康险（个人所得税）** | 个人购买符合规定的商业健康保险，扣除限额 **2400元/年**（200元/月），单位统一购买的视同个人购买 |

捐赠扣除

| 项目            | 扣除标准                                            |
| ------------- | ----------------------------------------------- |
| **一般公益性捐赠**   | 不超过年度利润总额 **12%** 的部分准予扣除；超过部分准予结转以后 **三年内** 扣除 |
| **捐赠住房作为公租房** | 在年度利润总额12%以内扣除，超过部分结转三年                         |
要点：
- 必须通过**公益性社会组织**或**县级以上人民政府及其部门**进行捐赠
- 公益性社会组织需依法取得**公益性捐赠税前扣除资格**
- 年度利润总额是指企业依照国家统一会计制度计算的大于零的数额
- **扣除顺序**：先扣除以前年度结转的捐赠支出，再扣除当年发生的捐赠支出
