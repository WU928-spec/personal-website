下列情形不得享受“500 万内免税、超过部分减半征收”
- **非居民企业**
- **非列举技术、非技术性收入**
- **未单独核算或核算不清**
- **关联交易未按独立交易原则定价**
- **禁止 / 限制出口的技术转让**

企业国际化的动因通常可以分为两大类：**外部动因**和**内部动因**。以下是具体分类和说明：

---

### **一、外部动因**
1. **市场因素**  
   - **寻求新市场**：国内市场饱和或增长缓慢，企业需要开拓海外市场以维持增长。  
   - **全球需求增长**：某些产品或服务在海外有更高的需求或增长潜力（如新兴市场）。  
   - **客户全球化**：客户业务扩展到海外，企业需要跟随客户以维持合作关系。

2. **竞争压力**  
   - **规避国内竞争**：国内市场竞争激烈，利润空间被压缩，企业通过国际化寻找蓝海市场。  
   - **应对全球竞争者**：若竞争对手已国际化，企业需跟进以保持竞争力。

3. **政策与贸易环境**  
   - **贸易壁垒降低**：区域贸易协定（如RCEP、USMCA）或WTO规则为企业提供便利。  
   - **政府激励**：东道国的税收优惠、补贴或基础设施支持吸引企业投资。  
   - **规避贸易限制**：通过海外设厂绕过关税、配额或制裁（如“关税跳跃型投资”）。

4. **技术因素**  
   - **技术扩散**：快速获取海外先进技术或管理经验（如通过并购或合作）。  
   - **产业升级需求**：通过国际化接触更高水平的技术标准或产业链。

---

### **二、内部动因**
1. **资源寻求**  
   - **获取低成本生产要素**：如劳动力、原材料、能源成本较低的地区（如制造业向东南亚转移）。  
   - **确保资源供应**：能源、矿产类企业通过投资海外保障资源稳定（如石油公司布局中东、非洲）。

2. **效率提升**  
   - **规模经济与范围经济**：通过全球化生产降低成本，优化供应链（如苹果的全球产业链布局）。  
   - **分散风险**：减少对单一市场的依赖，平衡汇率、政治或经济波动的影响。

3. **战略驱动**  
   - **品牌全球化**：提升国际知名度，强化品牌价值（如华为、三星的全球品牌战略）。  
   - **学习与创新**：通过国际化吸收先进知识，推动产品创新和管理优化。

4. **利润最大化**  
   - **利用核心能力**：将国内成功的技术、品牌或商业模式复制到海外，实现价值最大化。  
   - **税收优化**：通过国际布局合理避税（如部分企业在低税率国家设立总部）。

---

### **三、特殊动因**
- **跟随客户或伙伴**：如汽车零部件供应商随整车厂进入海外市场。  
- **被动国际化**：收到海外订单或询盘后顺势拓展业务。  
- **企业家精神**：企业领导者具备全球化视野，主动推动国际化。

国际化战略：

| 战略类型  | 全球协作 | 本土响应 | 核心逻辑        | 典型案例        |
| ----- | ---- | ---- | ----------- | ----------- |
| 国际战略  | 低    | 低    | 母国模式复制出海    | 早期星巴克、传统奢侈品 |
| 多国本土化 | 低    | 高    | 本地定制，区域独立   | 肯德基、联合利华    |
| 全球化战略 | 高    | 低    | 全球标准化，集中降本  | 苹果、特斯拉      |
| 跨国战略  | 高    | 高    | 全球协同 + 本土定制 | 海底捞、可口可乐    |

全球协作：总部对地方的控制越强，全球协作越高
本土响应：地方的权力越大，会针对地方进行特色改革本土响应越高


| 升级类型  | 核心逻辑     | 升级维度       | 典型特征          | 案例          |
| ----- | -------- | ---------- | ------------- | ----------- |
| 工艺升级  | 把事做得更好   | 同一产品、同一环节  | 优化生产流程、降本增效   | 工厂引入自动化生产线  |
| 产品升级  | 做更好的产品   | 同一环节、不同产品  | 推出高端 / 创新产品   | 从功能机升级为智能手机 |
| 功能升级  | 做更高价值的环节 | 同一价值链、不同环节 | 从制造向研发 / 品牌延伸 | 代工厂升级为自有品牌  |
| 价值链升级 | 换更高价值的赛道 | 跨价值链、跨行业   | 进入全新高附加值产业    | 电池厂升级为整车厂   |

功能升级：
- 不离开原有行业 / 价值链，仅向上游 / 下游拓展功能
- 核心是环节跨越，从「制造端」向「微笑曲线两端」攀升
价值链升级：
- 完全脱离原有行业 / 价值链，进入全新领域
