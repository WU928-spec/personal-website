风险是**客观存在、不确定何时发生、普遍伴随活动、但可以用概率测算、且随时代发展变化**的潜在损失来源。

| 特征       | 含义                         |
| -------- | -------------------------- |
| **客观性**  | 风险独立于人的意志而存在，不以人的主观愿望而消失   |
| **不确定性** | 是否发生、何时发生、损失程度均无法事先精确确定    |
| **普遍性**  | 无处不在、无时不有，任何经济活动都伴随风险      |
| **可测性**  | 大量同质风险可通过概率论和大数定律进行定量测算    |
| **发展性**  | 风险随社会、技术、经济环境变化而不断演变和产生新形态 |

风险三要素：风险因素+风险事件+风险后果
- **风险因素**：刹车缺陷、员工道德水平低、安保制度漏洞
    
- **风险事件**：交通事故、资金被侵占、货物被盗
    
- **风险后果**：人员伤亡、财务损失、停产
风险因素是造成风险的潜在原因，风险事件是造成风险的直接原因，最后产生的结果是风险后果

损失分类
	直接损失
		直接造成的损失：财产本身的损毁、人身伤害
	间接损失
		业务中断、声誉受损、市场份额下降、法律诉讼费
传统风险观和现代风险观

| 维度   | 传统风险观      | 现代风险观（市场经济/ERM） |
| ---- | ---------- | --------------- |
| 风险本质 | 纯粹的负面因素、危险 | 危险与机遇并存         |
| 结果认知 | 单一最可能结果    | 一系列可能结果的概率分布    |
| 应对态度 | 规避、消除      | 选择、管理、利用        |
| 可测性  | 可预测、可度量    | 可预测、可度量         |
