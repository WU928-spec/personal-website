# 项目复盘

## 背景

这次把个人网站从 Next.js 迁移到了 React + Vite，体验提升显著。

## 技术栈对比

| 维度 | Next.js | Vite + React |
|------|---------|-------------|
| 启动速度 | 慢 | 快 |
| 构建速度 | 中等 | 极快 |
| 部署复杂度 | 高 | 低 |
| 客户端路由 | 内置 | React Router |

## 关键决策

1. **放弃 SSR** — 个人网站不需要 SEO 优化到极致
2. **拥抱 SPA** — 路由切换更流畅
3. **Tailwind CSS** — 原子化样式开发效率极高

## 踩坑记录

### Vercel 404 问题

根目录缺少 `package.json` 和 `vercel.json` 导致子目录部署失败。解决：

```json
{
  "rewrites": [{ "source": "/(.*)", "destination": "/app/$1" }]
}
```

### TypeScript 严格模式

`tsconfig.app.json` 中 `exclude` 要加上 `"src/test"`，否则 vitest 类型会冲突。
