# 数字园艺的艺术

为什么我把网站当作花园而不是作品集——以及这如何改变了一切。

## 从作品集到花园

传统的个人网站像一个**作品集**：精心策划、完美呈现、从不改变。但数字花园不同：

- **种子**：零散的想法和笔记
- **萌芽**：初步的思考和草稿
- **成长**：持续迭代和完善
- **开花**：成熟的、可供分享的知识

## 我的园艺工具

| 工具 | 用途 |
|------|------|
| Obsidian | 本地知识库 |
| React + Vite | 网站构建 |
| Supabase | 数据持久化 |
| Vercel | 部署托管 |

## 为什么要公开生长？

> "知识在分享时生长，在封闭时枯萎。"

公开写作迫使我：
1. **更清晰地思考** — 写出来才能发现漏洞
2. **建立连接** — 与志同道合的人相遇
3. **获得反馈** — 他人的视角补充我的盲区
