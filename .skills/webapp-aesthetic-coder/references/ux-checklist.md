# UX 检查清单

## 通用检查项

### 1. 图片加载
- [ ] 所有 `<img>` 有 `alt` 属性
- [ ] 图片加载失败时有占位符（不是灰色方块或空白）
- [ ] 占位符设计美观（图标 + 颜色 + 首字）

### 2. 返回导航
- [ ] 每个页面有明确的返回上一页/返回首页路径
- [ ] 工具页面使用 `<BackToTools />` 组件
- [ ] 返回按钮位置显眼（左上角）

### 3. 加载状态
- [ ] 异步操作有 loading 指示（不只是在按钮上 disabled）
- [ ] 加载指示有文字说明（"正在搜索..." / "正在思考..."）
- [ ] 加载完成后自动通知用户（滚动/提示）

### 4. 错误处理
- [ ] API 失败时有明确错误提示（不是静默失败）
- [ ] 错误提示可关闭
- [ ] 错误后提供降级方案（如切换到本地模式）

### 5. 空状态
- [ ] 无数据时显示友好空状态（不是空白或报错）
- [ ] 空状态提供引导操作（如"输入关键词开始搜索"）

### 6. 滚动行为
- [ ] 用户主动滚动时，新内容不强制跳转
- [ ] 自动滚动只在适当时候触发（如 AI 回复完成后）
- [ ] 滚动位置合理（滚动到消息顶部，不是底部空 div）

### 7. 焦点管理
- [ ] 弹窗/模态框打开后焦点在内部
- [ ] 弹窗关闭后焦点回到触发元素
- [ ] Tab 键在弹窗内循环（焦点陷阱）

### 8. 键盘支持
- [ ] 所有按钮可用 Enter/Space 激活
- [ ] 输入框可用 Enter 提交
- [ ] 弹窗可用 Esc 关闭
- [ ] 导航可用方向键

## 工具页面专属检查项

### 9. 接入引导
- [ ] 没有 API Key 时明确提示用户如何获取
- [ ] 提供获取链接（如 SiliconFlow 免费注册）
- [ ] 已保存的 Key 有状态指示（在线/离线）

### 10. 状态指示
- [ ] 显示当前是否已接入 AI（在线/离线）
- [ ] 状态用视觉指示（绿色圆点 + 文字）
- [ ] 状态变化有动画反馈

### 11. 清空/重置
- [ ] 有清空对话/重置状态的按钮
- [ ] 清空前不二次确认（但可撤销）
- [ ] 清空后自动聚焦输入框

### 12. 历史保存
- [ ] 用户数据刷新后不丢失（localStorage）
- [ ] 历史数据加载失败时静默降级
- [ ] 清空历史时彻底删除（不残留）

### 13. 快速操作
- [ ] 提供常用快捷操作（快速回复、重新生成）
- [ ] 快速操作点击后直接执行（不是只填充输入框）
- [ ] 快速操作在适当时候隐藏（如已有对话时）

### 14. 详情展开
- [ ] 列表项可点击展开完整详情
- [ ] 详情弹窗有关闭按钮和点击外部关闭
- [ ] 详情包含所有必要信息（标题、描述、元数据、推荐理由）
