# Task 008: 博客卡片改为微博风格（文字在上，图片在下，真实图片）

## 背景
生活碎片页面的卡片目前使用假缩略图（blog-thumb-x.jpg），需要改为微博风格：文字在上，从正文中提取的真实图片在下。

## 文件
只改 `app/src/pages/Blog.tsx` 中的 `MomentCard` 组件

## 实现要求

1. **提取真实图片**：从 post.content 中用正则提取 `![](url)` 或 `![alt](url)` 中的所有图片 URL
2. **布局改为微博风**：文字内容（标题+摘要）在上方，图片在下方
3. **图片展示**：微博九宫格风格
   - 1张图：全宽 `aspect-[16/9]` 单图
   - 2-3张：flex row，等宽
   - 4张及以上：2列 grid
4. **删除假缩略图**：去掉 blog-thumb-*.jpg 占位图相关代码（img/onLoad/imageLoaded state 等）
5. **无图片的纯文字碎片**：只显示文字，无图片区

## 微博卡片布局参考

```
┌──────────────────────────────────┐
│  👤 Jasper ✨                    │

│  │          │          │          │

```
